📱 Installation Instructions (Android APK)

  1. Click on the Download Latest Version button.
  2. You will be redirected to the GitHub Release page.
  3. Download the .apk file from Assets section.
  4. Open the downloaded APK file on your device.
  5. If prompted, enable Install from Unknown Sources.
  6. Tap Install and wait for the installation to complete.
  7. Open ScanSplit and start using the app.

💻 Installation Instructions (PC)

  1. Click the download button to open the GitHub Release page.
  2. Download the setup file (.exe or .zip).
  3. Extract the file if it is in ZIP format.
  4. Run the setup or executable file.
  5. Follow on-screen instructions to complete installation.
  6. Launch ScanSplit from your desktop or start menu.

🛠️ System Requirements
